# 実装サンプルコード

このディレクトリには、Notion → ECモール連携の各種実装サンプルが含まれています。

## ディレクトリ構成

- `1-google-apps-script/` - Google Apps Scriptによる半自動化
- `2-zapier-templates/` - Zapier設定テンプレート
- `3-python-basic/` - Python基本実装
- `4-python-github-actions/` - GitHub Actions自動実行版
- `5-csv-converter/` - CSV変換ツール
- `6-serverless/` - サーバーレス実装例

## 推奨される導入順序

1. **まず試す**: `1-google-apps-script/`
2. **次に**: `2-zapier-templates/` または `4-python-github-actions/`
3. **最終形**: `3-python-basic/` または `6-serverless/`
